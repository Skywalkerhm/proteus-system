## 📝 技术说明

### 记忆系统设计

```
Working Memory (工作记忆)
├── 存储：当前任务上下文
├── 生命周期：任务期间
└── 用途：Agent 间通信、临时状态

Episodic Memory (场景记忆)
├── 存储：完整任务执行轨迹
├── 生命周期：永久
└── 用途：复盘学习、案例参考

Semantic Memory (语义记忆)
├── 存储：Agent 画像、任务模式、规则库
├── 生命周期：永久 + 持续进化
└── 用途：决策支持、模式匹配
```

### Hub 工作流

```
1. receive_task() → 2. parse_task() → 3. form_claw() → 4. execute_task() → 5. deliver_task()
      ↓                    ↓                 ↓                ↓                 ↓
   记录任务            匹配模式           组建团队          执行任务          交付 + 学习
```
