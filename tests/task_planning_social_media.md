## 🎯 任务分析

### 核心需求拆解

| 维度 | 分析 |
|------|------|
| **目标受众** | 创业团队（资源有限、需要高效内容） |

3. ✅ **平台适配** - 不同平台的文案风格和内容形式不同
4. ✅ **互动设计** - 包含明确的 CTA 和互动引导
5. ✅ **灵活空间** - 预留热点跟进的调整空间

### 3. Agent 匹配算法

```python
# 简化版匹配逻辑
def match_agent(required_skills, agent_profile):
    agent_skills = set(agent_profile["skills"])
    required = set(required_skills)
    
    # 计算技能覆盖率
    coverage = len(agent_skills & required) / len(required)
    
    # 考虑历史成功率
    success_rate = agent_profile["stats"]["success"] / max(agent_profile["stats"]["total"], 1)
    
    # 综合评分
    score = coverage * 0.7 + success_rate * 0.3
    
    return score
```

## 📋 执行计划

### 阶段 1: 调研分析 (45min)

**负责**: Research Agent

```
输入: 创业团队信息（行业、产品、目标用户）
       ↓
1. 目标受众画像
   - 年龄、性别、地域分布
   - 兴趣标签、痛点需求
   - 社交媒体使用习惯
       ↓
2. 竞品分析
   - 同类账号内容类型
   - 高互动内容特征
   - 发布频率和时段
       ↓
3. 趋势搜集
   - 行业热点话题
   - 平台流行形式
   - 可借势的节点
       ↓
输出：受众画像文档 + 竞品分析报告
```

### 阶段 2: 内容策划 (30min)

**负责**: Content Agent

```
输入: 调研结果
       ↓
1. 主题规划
   - 7 天内容主题（不重复）
   - 内容类型分布（故事/产品/互动/干货）
   - 品牌信息植入点
       ↓
2. 发布节奏
   - 每日发布时段（考虑用户活跃时间）
   - 平台选择（小红书/微博/抖音）
   - 发布频率（1-2 条/天）
       ↓
3. 互动设计
   - 每日 CTA 设计
   - 评论回复策略
   - 引导分享机制
       ↓
输出：内容主题日历
```

### 阶段 3: 文案创作 (90min)

**负责**: Content Agent

```
输入: 内容主题日历
       ↓
逐天创作:
├── Day 1: 创业故事 - 初心篇
│   ├── 标题：吸引点击
│   ├── 正文：300-500 字，有情感共鸣
│   ├── 标签：#创业日常 #初心
│   └── CTA: 评论区分享你的故事
├── Day 2: 产品亮点 - 功能解析
├── Day 3: 用户案例 - 真实反馈
├── Day 4: 团队日常 - 幕后故事
├── Day 5: 干货分享 - 行业洞察
├── Day 6: 互动话题 - 用户参与
└── Day 7: 周末轻松 - 品牌人格化
       ↓
输出：7 天文案草稿（含标题、正文、标签、CTA）
```

### 阶段 4: 视觉建议 (60min)

**负责**: Content Agent

```
输入: 文案草稿
       ↓
1. 视觉风格定义
   - 色调（品牌色 + 辅助色）
   - 字体（标题字体 + 正文字体）
   - 图片风格（实拍/插画/混合）
       ↓
2. 逐天配图建议
   - Day 1: 团队合照 + 办公室场景
   - Day 2: 产品功能对比图
   - Day 3: 用户反馈截图
   - ...
       ↓
3. 设计资源推荐
   - 免费图库（Unsplash、Pexels）
   - 设计工具（Canva、稿定设计）
   - 模板参考
       ↓
输出：视觉风格指南 + 配图建议清单
```

### 阶段 5: 质量审核 (30min)

**负责**: Review Agent

```
输入: 完整内容方案
       ↓
质量检查清单:
├── ✓ 7 天内容无重复主题
├── ✓ 每条文案有明确目标
├── ✓ 配图建议清晰可执行
├── ✓ 包含互动引导
├── ✓ 符合品牌调性
├── ✓ 无错别字和语法错误
├── ✓ CTA 设计合理
└── ✓ 发布时段科学
       ↓
优化建议:
- 文案调整建议
- 视觉优化建议
- 发布策略调整
       ↓
输出：审核报告 + 优化建议
```

## 🔄 协作协议

### Agent 间通信规则

```
Hub → Research Agent: "请执行受众调研，输出画像文档"
Research Agent → Hub: "调研完成，发现目标用户主要是 25-35 岁女性"
Hub → Content Agent: "基于调研结果，请策划 7 天内容主题"
Content Agent → Hub: "主题日历已完成，请审核"
Hub → Review Agent: "请对完整方案进行质量审核"
Review Agent → Hub: "审核通过，建议优化 Day3 的 CTA"
Hub → Content Agent: "请根据审核意见优化 Day3"
Content Agent → Hub: "优化完成"
Hub → User: "交付最终方案"
```

### 异常处理

| 异常类型 | 处理流程 |
|---------|---------|
| 调研数据不足 | Research Agent 请求 Hub 补充信息 |
| 文案创作困难 | Content Agent 请求 Research Agent 补充案例 |
| 审核不通过 | Review Agent 列出具体问题，Content Agent 修改 |
| 时间超限 | Hub 介入调整优先级或延长截止时间 |
